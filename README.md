# MDB
A movies database site where a user can review his/her favourite movies.

## Design inspiration
Netflix, Rotten Tomatoes and IMDB

## Tools Used
- **Forntend**: HTML, CSS, JS
- **Server Side**: Node.js, Express
- **Database**: mysql
- **OS**: Ubuntu
